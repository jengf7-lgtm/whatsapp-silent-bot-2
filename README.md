# 🛡️ WhatsApp Silent Bot (ระบบป้องกันสแปมกลุ่ม)

บอทจัดการกลุ่ม WhatsApp อัจฉริยะที่ช่วยป้องกันการสแปมลิงก์และการส่งรูปภาพรัวๆ เพื่อให้กลุ่มของคุณปลอดภัยและเป็นระเบียบเรียบร้อยตลอด 24 ชั่วโมง

## ✨ คุณสมบัติหลัก (Key Features)

* 🛑 **ระบบกรองและลบลิงก์อัตโนมัติ (Anti-Link Check):** ลบลิงก์ (HTTP/HTTPS/WWW) ที่สมาชิกส่งทันที และเตะผู้ส่งออกจากกลุ่มทันที (ยกเว้นแอดมิน)
* 📷 **ระบบจำกัดและบล็อกการส่งรูปภาพรัว (Anti-Image Spam):**
  * สมาชิกส่งรูปได้ไม่เกิน 2 ใบ (ระบบจะลบเพื่อความสะอาดของกลุ่ม)
  * หากส่งใบที่ 3 ขึ้นไป ระบบจะบล็อกผู้ใช้งานทันที!
  * **โหมดล้างบางสมบูรณ์แบบ (Perfect Shield Mode):** ล็อกกลุ่มให้เฉพาะแอดมินส่งข้อความได้ชั่วคราว + เตะผู้สแปมออก + เปิดโหมดกวาดล้างรูปที่หลุดรอดเข้ามาเป็นเวลา 5 วินาที
  * ปลดล็อกกลุ่มให้กลับมาคุยกันได้ปกติอัตโนมัติหลังจากผ่านไป 1 นาที (60 วินาที)
* ⚡ **ระบบทำงานอัตโนมัติ 100%:** ไม่จำเป็นต้องมีคำสั่งควบคุม (Zero Command)

## 🛠️ ความต้องการของระบบ (Prerequisites)

* **Node.js** เวอร์ชัน 16 ขึ้นไป
* บัญชี WhatsApp สำหรับทำหน้าที่เป็นบอท (ต้องตั้งค่าให้บัญชีนี้เป็น **Admin (แอดมิน)** ของกลุ่มที่ต้องการดูแล)

---

## 🚀 วิธีติดตั้งและใช้งาน (Installation & Setup)

1. **โคลนโปรเจกต์นี้ลงบนเครื่องของคุณ:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/whatsapp-silent-bot.git
   cd whatsapp-silent-bot
   ```

2. **ติดตั้ง Dependencies:**
   ```bash
   npm install
   ```

3. **เริ่มการทำงานบอท:**
   ```bash
   node index.js
   ```

4. **สแกน QR Code:**
   * เมื่อรันสำเร็จ จะมี QR Code แสดงขึ้นมาบน Terminal
   * เปิดแอป WhatsApp ในมือถือ -> ไปที่ **Linked Devices (อุปกรณ์ที่เชื่อมโยง)** -> กด **Link a Device** แล้วสแกน QR Code นี้
   * เมื่อเชื่อมต่อเสร็จแล้ว บอทจะเริ่มทำงานทันที!

## 📂 โครงสร้างโปรเจกต์ (Project Structure)

* `index.js` - โค้ดการทำงานหลักของระบบตรวจจับและลบข้อความสแปม
* `package.json` - ไฟล์ตั้งค่า Dependencies ของโปรเจกต์
* `.gitignore` - ตั้งค่าไม่ให้อัปโหลดข้อมูลเซสชัน WhatsApp ที่สำคัญ (`.wwebjs_auth`, `.wwebjs_cache`) และโฟลเดอร์ `node_modules` ขึ้น GitHub

---

## 🛡️ ความปลอดภัยของข้อมูล (Security Note)

> [!WARNING]
> โฟลเดอร์ `.wwebjs_auth` และ `.wwebjs_cache` ที่ถูกสร้างขึ้นหลังจากการสแกน QR Code **มีข้อมูลการยืนยันตัวตนและการเข้าถึงบัญชี WhatsApp ของคุณ**
>
> ห้ามแชร์โฟลเดอร์เหล่านี้ให้กับผู้อื่น และตัวโปรเจกต์นี้ได้รับการตั้งค่า `.gitignore` เพื่อป้องกันไม่ให้อัปโหลดโฟลเดอร์เหล่านี้ขึ้น GitHub เรียบร้อยแล้ว เพื่อความปลอดภัยสูงสุดของคุณ
